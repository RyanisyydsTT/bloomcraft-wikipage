自訂附魔系統

這裡的附魔，不再只是原版 Minecraft 那些常見的「耐久」「保護」「效率」。 你將能體驗到 數百種全新設計的附魔效果，包含戰鬥強化、工具加成、防禦守護、甚至特殊能力。

系統特色

超過 300 種附魔，依照稀有度分為：基礎、獨特、精英、終極、傳奇、神話。 多樣獲取方式：附魔台 /enchanter、修補匠 /tinkerer、煉金術師 /alchemist、活動掉落等。 裝備專屬性：每個附魔都有指定可附加的裝備類型（武器、盔甲、工具、弓弩）。 成功, 摧毀率：附魔書應用時存在成功率, 失敗風險，建議搭配 白色卷軸 保護。 合成, 強化：相同等級的附魔書可合成提升等級，還能透過各種卷軸、魔法之塵來提高成功率。

🎯 新手應該怎麼開始？

  1. 使用 /enchanter 購買一個附魔書。
  2. 拿到的附魔書，直接拖曳到裝備上即可使用。
  3. 注意書本上的 成功率 / 損壞率，必要時準備 白卷軸 保障裝備安全。
  4. 隨著等級, 裝備提升，可以逐步挑戰 更高稀有度的附魔。

小提醒

別忽視低階附魔，許多效果能, 高階附魔疊加，發揮驚人力量。 若裝備有「不可修改」標記，則無法套用自訂附魔。 物品的靈魂數量可以使用靈魂追蹤器查看
